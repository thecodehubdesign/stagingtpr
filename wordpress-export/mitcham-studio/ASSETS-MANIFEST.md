## Download Instructions

1. Open the preview URL: `https://stagingtpr.lovable.app/studios/mitcham`
2. Right-click on images → "Save Image As"
3. For videos, use browser developer tools (Network tab) or video download extensions
4. Organize files according to the folder structure above
5. Replace placeholder image paths in index.html if filenames differ

---

## WordPress Integration Notes

1. **Upload all assets to WordPress Media Library**
2. **Update paths in HTML/CSS** to use WordPress media URLs
3. **Google Maps:** Add API key via WordPress customizer or theme options
4. **Vimeo:** Embed works without changes
5. **Fonts:** Consider using a WordPress plugin for Google Fonts or enqueue in functions.php

```php
// Example: Enqueue Montserrat in WordPress
function tpr_enqueue_fonts() {
    wp_enqueue_style('montserrat-font', 'https://fonts.googleapis.com/css2?family=Montserrat:wght@400;500;600;700&display=swap', array(), null);
}
add_action('wp_enqueue_scripts', 'tpr_enqueue_fonts');
```
